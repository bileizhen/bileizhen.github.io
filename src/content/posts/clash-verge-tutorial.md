## 1. 电脑端：Clash Verge Rev 食用指南

### 1.1 软件下载：选对“座驾”

别再去搜那些乱七八糟的下载站了，满屏的广告看了就头大。咱们直接去 GitHub 搬救兵：

*   **项目地址**：[Clash Verge Rev GitHub](https://github.com/clash-verge-rev/clash-verge-rev)
*   **怎么选？** 进到 Releases 页面：
    *   **Windows 用户**：找 `Clash.Verge_x.x.x_x64_zh-CN.msi`（安装版）或者 `..._Portable.zip`（解压即用）。
    *   **Mac 用户**：找 `.dmg` 后缀的文件。
*   **注意**：一定要认准 **Rev** 后缀，这是目前社区维护的“正统续作”，功能最稳。

### 1.2 核心步骤：手把手起飞

安装完软件打开，你会看到一个非常 Modern 的界面。别被那些菜单吓到，跟着我一步步点：

#### 第一步：导入配置 (Profiles)
1.  去你的机场后台，点那个 **“一键订阅”** 里的 **“复制订阅地址”**（别选错了，要 URL 链接）。
2.  回到 Clash Verge，点击左侧菜单栏的 **“配置 (Profiles)”**。
3.  在顶部的输入框里，直接 **Ctrl+V** 粘贴链接，然后点击旁边的 **“导入 (Import)”**。
4.  **关键点**：导入成功后，下面会出现一个配置卡片。**用鼠标左键点一下它**，看到卡片边框变色了（通常是蓝色或紫色），这才算生效。

#### 第二步：开启系统代理 (System Proxy)
1.  点击左侧菜单最顶上的 **“设置 (Settings)”**。
2.  找到 **“系统设置”** 板块下的 **“系统代理 (System Proxy)”**。
3.  **点亮那个开关！** 变成彩色就说明你的电脑流量已经开始走代理了。
4.  （可选）建议同时点亮 **“开机自启动”**，下次开机就不用手动开了。

#### 第三步：选择节点 (Proxies)
1.  点击左侧的 **“代理 (Proxies)”**。
2.  你会看到一堆节点组。通常第一个是 **“Proxy”** 或 **“节点选择”**。
3.  建议先选 **“自动选择 (Auto)”**，或者手动点一下那个“小闪电”图标测速，选个延迟数字最小的（比如香港、新加坡）。
4.  现在打开 Google 或 YouTube，如果能丝滑加载，恭喜你，电脑端通关！🎉

## 2. 手机端：NekoBox (Android) 极简教程

电脑整好了，手机也得安排上。**NekoBox**（又叫猫盒）是目前 Android 端最推荐的工具，支持协议全，还免费开源。

### 2.1 软件下载
*   **GitHub 地址**：[NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid)
*   **Play 商店**：直接搜 `NekoBox` 也能找到。
*   **下载哪个？** 现在的手机一般选 `NB4A-x.x.x-arm64-v8a.apk`。

### 2.2 核心步骤：一分钟搞定
1.  **复制链接**：同样去机场后台，复制那个 **“Clash 订阅链接”** 或 **“通用订阅链接”**。
2.  **导入订阅**：
    *   打开 NekoBox，点击右上角的 **“+”** 号。
    *   选择 **“订阅 (Subscription)”**。
    *   随便起个名字（比如：我的魔法），在 **“分组链接”** 里粘贴刚才复制的地址。
    *   点击右上角的 **“确定/保存”**。
3.  **获取节点**：
    *   回到主界面，点击右上角的三个点（或者侧边栏），选择 **“更新订阅”**。
    *   这时候你会看到一长串节点冒了出来。
4.  **开始连接**：
    *   随便选一个延迟低的节点点一下。
    *   点击屏幕右下角那个 **“圆圈图标”**（播放键样式）。
    *   第一次连接会弹出“连接请求”，点 **“确定”**。
    *   **图标变绿**，搞定！

## 3. 避坑指南：为什么我还是打不开？

新手最容易掉进的三个坑，老司机提前给你填平：

1.  **开了代理还是上不去？**
    *   **查时间**：检查电脑/手机时间是否为“自动同步”！魔法对时间非常敏感，差个 1 分钟可能就握手失败。🕰️
    *   **查流量**：去机场后台看看，是不是流量用光了，或者会员过期了。😅
2.  **部分软件不走代理？**（针对 PC）
    *   有些倔强的软件（比如 UWP 应用或某些游戏）不听话。去设置里把 **“TUN 模式”** 打开。这玩意儿是内核级的，管它愿不愿意，统统拉走！
3.  **手机网速慢？**（针对 NekoBox）
    *   在 NekoBox 设置里，找到 **“路由设置”**，确保开启了 **“绕过局域网和中国大陆”**。这样刷抖音、用微信就不会被减速了。
